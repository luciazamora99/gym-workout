# Gym Workout Web App

Frontend web application developed to plan weekly gym workouts and view workout routines by day.

## Features
- Weekly workout schedule
- Routine pop-ups by day
- Recipe section with pop-up details
- Smooth navigation between pages
- Current day highlighted

## Tech Stack
- HTML
- CSS
- JavaScript
